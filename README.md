# DDoS Attack Detection using Semi-Supervised Learning

This is an academic machine learning project focused on detecting DDoS attacks from network traffic data. The objective of the project is to classify traffic as normal traffic or DDoS attack traffic by analyzing network traffic patterns and applying machine learning techniques.

## Domain
Artificial Intelligence / Machine Learning with network security use case.

## Tools and Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Jupyter Notebook / Google Colab.

## Techniques Used
Entropy Estimation, Co-Clustering, Extra Trees Algorithm, Semi-Supervised Learning, Feature Analysis, Model Training, Testing, and Performance Evaluation.

## My Role
I worked on understanding the dataset, basic preprocessing, feature analysis, model training/testing support, and result evaluation using performance metrics.

## Team Size
3–4 members.

## Project Complexity
The project had moderate complexity because it involved network traffic analysis, labelled/unlabelled data handling, semi-supervised learning concepts, and model performance evaluation.
